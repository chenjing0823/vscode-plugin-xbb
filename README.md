# vscode-plugin-xbb
销帮帮web端国际化插件
公司自用定制化功能
# 功能
1. 国际化的推进很多代码阅读性的语言较差，可以用来代码阅读
2. 可以用来当辅助工具，检索替换检索校验。用来验证国际化工作是否正确
